# Relatório Técnico - Bella Essência

## 📌 Objetivo
Landing page para conversão de clientes em uma clínica de estética, com foco em:  
- Design responsivo  
- Integração com WhatsApp  
- Performance otimizada

## 🛠 Tecnologias Utilizadas
- **Front-end**: HTML5, CSS3, JavaScript (Vanilla)  
- **Ferramentas**: Font Awesome, Formspree (simulado), Google Fonts  
- **Otimização**: Compressão de imagens, Mobile-first  

## ⚙️ Funcionalidades-chave
1. **Carrossel interativo** de procedimentos  
2. **Menu mobile** com animação  
3. **Botão flutuante** de WhatsApp  
4. **Layout adaptável** (testado em 3 dispositivos)  

## 📊 Métricas de Performance
(Adicione prints do Lighthouse)
- Performance: 92/100  
- Acessibilidade: 95/100  
- SEO: 89/100
